## Quick Reference

Sorty provides comprehensive keyboard shortcuts for efficient navigation and operation. All shortcuts follow macOS conventions.

<Info>
  **Symbols**: ⌘ (Command), ⇧ (Shift), ⌥ (Option), ⌃ (Control), ⏎ (Return), ⎋ (Escape)
</Info>

## Navigation Shortcuts

Quickly jump between different views in Sorty.

| Shortcut | Action | Description |
|----------|--------|-------------|
| <kbd>⌘1</kbd> | Go to Organize | Main organization view |
| <kbd>⌘2</kbd> | Go to Workspace Health | Health monitoring dashboard |
| <kbd>⌘3</kbd> | Go to Duplicates | Duplicate file detection |
| <kbd>⌘4</kbd> | Go to Exclusions | Manage exclusion rules |
| <kbd>⌘5</kbd> | Go to Watched Folders | Watched folder management |
| <kbd>⌘,</kbd> | Open Settings | Application preferences |
| <kbd>⇧⌘H</kbd> | Open History | View organization history |
| <kbd>⇧⌘L</kbd> | Open The Learnings | Learning dashboard |
| <kbd>⌘\\</kbd> | Toggle Sidebar | Show/hide navigation sidebar |
| <kbd>⌘?</kbd> | Sorty Help | Open help documentation |

<Tip>
  **Pro tip**: Use <kbd>⌘1</kbd>-<kbd>⌘5</kbd> to quickly switch between main views without touching the mouse.
</Tip>

## File Operations

Manage files and sessions efficiently.

| Shortcut | Action | Description |
|----------|--------|-------------|
| <kbd>⌘N</kbd> | New Session | Clear current state and start fresh |
| <kbd>⌘O</kbd> | Open Directory | Select a folder to organize |
| <kbd>⌘E</kbd> | Export Results | Export organization plan (JSON, CSV, HTML) |
| <kbd>⌘A</kbd> | Select All Files | Select all files in current view |
| <kbd>⌘Z</kbd> | Undo | Revert last organization operation |

<Accordion title="What does 'New Session' do?">
  Starting a new session (<kbd>⌘N</kbd>) clears the current organization state, resets the preview, and allows you to start fresh with a new folder. It does **not** delete any files or history.
</Accordion>

## Organization Operations

Control the AI organization process.

| Shortcut | Action | Description |
|----------|--------|-------------|
| <kbd>⌘R</kbd> | Start Organization | Begin AI analysis of selected folder |
| <kbd>⇧⌘R</kbd> | Regenerate Organization | Re-run analysis with current settings |
| <kbd>⌘⏎</kbd> | Apply Changes | Execute the proposed organization |
| <kbd>⇧⌘P</kbd> | Preview Changes | Preview what will change before applying |
| <kbd>⎋</kbd> | Cancel Operation | Stop the current operation |

### Organization Workflow

<Steps>
  <Step title="Select folder">
    Press <kbd>⌘O</kbd> to choose a directory
  </Step>
  <Step title="Start organization">
    Press <kbd>⌘R</kbd> to begin AI analysis
  </Step>
  <Step title="Review results">
    Use <kbd>⇧⌘P</kbd> to preview changes
  </Step>
  <Step title="Apply changes">
    Press <kbd>⌘⏎</kbd> to execute the organization
  </Step>
  <Step title="Undo if needed">
    Press <kbd>⌘Z</kbd> to revert immediately
  </Step>
</Steps>

## The Learnings Shortcuts

Manage your learning profile and preferences.

| Shortcut | Action | Description |
|----------|--------|-------------|
| <kbd>⇧⌘L</kbd> | Open Dashboard | Open The Learnings dashboard |
| <kbd>⌥⌘H</kbd> | Start Honing Session | Begin preference refinement Q&A |

### Learnings Dashboard Navigation

Once inside The Learnings dashboard:

- **Tab**: Navigate between Overview, Preferences, and Activity tabs
- **Space**: Expand/collapse activity items
- **⏎**: Open detailed view for selected item
- **⌘F**: Search learning data

<Info>
  Honing sessions (<kbd>⌥⌘H</kbd>) help refine your organization preferences through AI-generated questions based on your recent activity.
</Info>

## Custom Keyboard Shortcuts (macOS)

You can create custom keyboard shortcuts for any menu command in macOS System Settings.

<Steps>
  <Step title="Open System Settings">
    Go to **System Settings → Keyboard → Keyboard Shortcuts**
  </Step>
  <Step title="Select App Shortcuts">
    Click **App Shortcuts** in the sidebar
  </Step>
  <Step title="Add shortcut">
    Click the **+** button and select **Sorty**
  </Step>
  <Step title="Configure">
    Enter the exact menu command name and assign your preferred shortcut
  </Step>
</Steps>

<Warning>
  Custom shortcuts must not conflict with existing system or app shortcuts. macOS will warn you if there's a conflict.
</Warning>

## Keyboard-First Workflow

Complete an organization without touching the mouse:

```
⌘O        → Open directory picker
⏎         → Confirm selection
⌘R        → Start organization
[wait]    → AI analyzes files
⇧⌘P       → Preview changes
↑/↓       → Review folders
Space     → Expand/collapse details
⌘⏎        → Apply changes
⇧⌘H       → View in history
```

<Tip>
  Master these shortcuts to organize files 10x faster. The keyboard-first workflow becomes muscle memory after a few uses.
</Tip>
